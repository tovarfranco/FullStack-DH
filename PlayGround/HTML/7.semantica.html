Etiquetas Semanticas: permite generar un codigo mas entendible, para humanos como para los buscadores. Las que veremos, son todas etiquetas de bloques Ejemplo: Header:
sirve para la cabecera del documento, o la cabecera de una seccion de contenido. va el logotipo, barra de navegacion, enlaces a redes sociales, campo de busquedas. Nav:
sirve para barra de navegación. Sea principal o alternativa. se suele implementar listas e hipervinculos. Footer:
sirve para generar el pie de pagina. derechos reservados, enlaces a webs. Section:
define seccion del contenido. Si quisieramos crear un breve apartado de un producto, seria buena eleccion. tambien podemos usar header y footer dentro de éstas. EJ:

Article:
permite definir pieza de contenido independiente, funcionar por si solo si necesidad del contenido que lo rodea: producto, servicio, noticia. Generalmente lo vemos dentro de secciones.
